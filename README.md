# Lazada Style Guides

Built with Hugo and Wercker

## Dependencies

- Git
- Brew
- Hugo

### Install Hugo

```
brew install hugo
```


## Run development environment

```
hugo serve
```

## Deploy

```
git add [files]
git commit -m "Description of the changes"
git push origin master
```
